Ajout d'une nouvelle traduction
-------------------------------

* Ajouter la chaine dans functions.php
* Dans l'admin de WP (Réglages - Langues - Traduction de phrases), ajouter la chaine
* Most of translations strings are managed in 'functions.php' where they are registered.
Translations are then made in the admin panel of WP (Settings - Languages - Strings translations )
